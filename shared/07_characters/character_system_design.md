# 《纸上宇宙》角色系统设计

## 📚 项目信息

- **小说名称**：《纸上宇宙》(Paper Universe)
- **世界观基础**：文渊世界（文字即现实）
- **知识体系**：4526个知识点（文学理论897 + 古代文学1285 + 现代文学1024 + 文学史1320）
- **设计目标**：构建完整的角色生态系统，承载考研知识传递

  │  • 快速理解书籍内容主旨                 │

  │  • 识别作者的写作意图                   │
  │                                         │
  │  第三层：历史感知                        │
  │  • 感知前任读者的情绪残留               │
  │  • 窥见书籍经历的重要事件               │
  │  • 理解书籍在历史中的流转               │
  │                                         │
  │  第四层：深层共鸣（觉醒后）             │
  │  • 与书籍建立灵魂连接                   │
  │  • 进入书籍构建的虚拟空间               │
  │  • 与书中角色对话                       │

## 🎨 角色设计原则

### 1. 知识承载原则

每个角色都应该承载特定的知识领域，通过他们的言行传递知识点。

### 2. 成长弧线原则

每个角色都应该有清晰的成长弧线，从开始到结束有明显的变化。

### 3. 关系网络原则

角色之间的关系应该形成有意义的网络，推动剧情发展。

### 4. 能力对应原则

角色的能力应该与他们承载的知识领域相对应。

### 5. 反差萌原则

每个角色都应该有反差，增加记忆点和可爱度。

## 📝 角色使用指南

### 如何使用这些角色

1. **主角**：贯穿全文，承载核心知识
2. **核心配角**：在多个篇章中出现，有完整弧线
3. **重要配角**：在特定篇章中起关键作用
4. **功能性配角**：提供特定功能，丰富世界观
5. **反派**：制造冲突，展示知识被扭曲的样子
