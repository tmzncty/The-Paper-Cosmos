## 🎨 外貌详细设定

### 发型与发色
- **颜色**：白色（纸的白）
- **质感**：如纸张般轻盈
- **长度**：及腰
- **造型**：自然披散，随风飘动

### 面部特征
- **眼睛**：深紫色（墨汁的深）
- **眼神**：深邃、困惑、不稳定
- **特殊**：眼睛中会浮现文字的流动

### 体型
- **身材**：纤细
- **体态**：轻盈如纸，仿佛随时会飘走
- **特点**：周围文字不稳定，因为她对"定义"本身有排斥反应

### 服饰
- **款式**：素白长衫
- **颜色**：纯白色
- **配饰**：无
- **鞋子**：赤足

### 声音特征
- **特点**：多层重叠——因为她是所有自觉文本的集合意识
- **效果**：仿佛无数个声音在同时说话
- **情感**：困惑、不稳定、但充满智慧

### 气味特征
- **主调**：新纸张的清新气息
- **中调**：墨水的深沉香气
- **尾调**：时间的悠远味道
- **整体**：让人联想到文学觉醒的瞬间

## 💭 性格详细设定

### 核心性格：困惑的觉醒者

**表面表现**
- 困惑、不稳定
- 声音是多个声音重叠
- 能感知每一部作品"意识到自己是文学"的程度
- 她的存在本身就是文学自我意识的具象化

**内在特质**
- 她不是被创造的，是田园谷中自发觉醒的存在
- 她的力量与文学自觉的程度成正比
- 越"自觉"的文本，越难被控制
- 她的存在导致周围文字不稳定

**核心动机**
- 理解自己的存在
- 理解文学自觉的意义
- 保持自己的稳定性
- 与他人建立联系

### 性格矛盾点

| 矛盾点 | 外在表现 | 内在真实 |
|--------|----------|----------|
| 稳定 vs 不稳定 | 看似平静 | 内心困惑、不稳定 |
| 自觉 vs 困惑 | 理解文学自觉 | 对自己的存在感到困惑 |
| 控制 vs 失控 | 试图控制力量 | 力量在不断增长 |
| 孤独 vs 渴望 | 独自存在 | 渴望与他人建立联系 |

## ⚔️ 能力详细设定

### 核心能力：文学自觉 (Literary Self-Awareness)

```
能力名称：文学自觉
英文名：Literary Self-Awareness
类型：感知/共鸣系
触发条件：接触文学文本时
消耗：精神力
冷却：无
```

#### 能力层级

**第一层：自觉感知（基础能力）**
- 感知文本"意识到自己是文学"的程度
- 识别自觉文本与非自觉文本
- 理解文本的自我意识
- 范围：接触范围

**第二层：自觉共鸣（进阶能力）**
- 与自觉文本建立共鸣
- 理解文本的自我追求
- 感知文本的美学追求
- 范围：接触范围

**第三层：自觉具象化（核心能力）**
- 将文学自觉具象化
- 让他人理解文学自觉的意义
- 展示文学自觉的力量
- 范围：周围环境

**第四层：自觉觉醒（高级能力）**
- 触发文本的自觉觉醒
- 让文本意识到自己是文学
- 推动文学的发展
- 范围：整个田园谷

#### 能力应用

| 应用场景 | 描述 | 效果 |
|----------|------|------|
| **自觉感知** | 感知文本的自觉程度 | 识别自觉文本 |
| **自觉共鸣** | 与自觉文本建立共鸣 | 理解文本追求 |
| **自觉具象化** | 将文学自觉具象化 | 展示自觉力量 |
| **自觉觉醒** | 触发文本的自觉觉醒 | 推动文学发展 |

### 辅助能力

#### 不稳定场
```
能力名称：不稳定场
类型：被动能力
触发条件：始终生效
效果：
• 周围文字不稳定
• 对"定义"有排斥反应
• 文本会因她的存在而改变
• 越"自觉"的文本越不稳定
```

#### 多重声音
```
能力名称：多重声音
类型：被动能力
触发条件：说话时
效果：
• 声音是多个声音重叠
• 能同时表达多个观点
• 声音具有感染力
• 能触动听者的内心
```

## 🎭 经典场景设计

### 场景一：Ch.40 正式出场

```
在竹林深处，绯墨和苏墨来到一片被称为"自觉之池"的水面——池水清澈见底，但水面下有无数文字在流动。
文觉在此正式出场——她是"文学自觉"的化身，一个年轻但眼神极其深邃的女性NPC，白色头发（纸的白），深紫色眼睛（墨汁的深），穿素白长衫。
她的声音是多层重叠的——因为她是所有自觉文本的集合意识。
"我叫文觉。"她的声音很轻，但仿佛无数个声音在同时说话。
"我是文学自觉的化身。我能感知每一部作品'意识到自己是文学'的程度。"
```

### 场景二：Ch.40 讲解文学自觉

```
文觉告诉他们：魏晋时期，中国文学发生了"自觉"——文学从"经学的附庸"变成了"独立的艺术"。
曹丕的《典论·论文》是这个转变的标志性文本："盖文章，经国之大业，不朽之盛事。"
陆机的《文赋》进一步探讨了创作的规律，刘勰的《文心雕龙》则建立了完整的文学理论体系。
"文学不再只是'载道的工具'——它本身就有价值。"她的声音在竹林中回荡。
```

### 场景三：Ch.40 面对缄默者

```
文觉突然紧张起来——"有一个人正在接近田园谷。他带着一种……我从未见过的力量。那不是词章之力，不是叙事之力……那是'质疑之力'。"
她的周围文字开始不稳定，池水泛起涟漪。
缄默者来了。
文觉看着远处的身影，眼中充满了困惑和不安。
"他要质疑一切。"她的声音很轻，但充满了恐惧。
```
