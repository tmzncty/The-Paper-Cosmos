## 🎨 外貌详细设定

### 发型与发色
- **颜色**：墨色丝线（如竹简的墨迹）
- **质感**：如丝线般光滑
- **长度**：及腰
- **造型**：用竹简编成的发饰束起

### 面部特征
- **眼睛**：两面微型目录页（眼睛本身就是目录）
- **眼神**：精确、冷静、带着目录学的严谨
- **特殊**：眼睛会随着目录的变化而变化

### 体型
- **身材**：中等
- **体态**：挺拔优雅，动作如目录般精确
- **特点**：手指修长，是翻阅目录的痕迹

### 服饰
- **款式**：竹简编成的衣裙
- **颜色**：竹青色为主，点缀墨色
- **配饰**：竹简编成的发饰、腰带
- **鞋子**：竹简编成的凉鞋

### 气味特征
- **主调**：竹子的清新气息
- **中调**：墨水的苦涩香气
- **尾调**：时间的悠远味道
- **整体**：让人联想到古老的目录体系

## 💭 性格详细设定

### 核心性格：精确的守护者

**表面表现**
- 用极精确的语言解释问题
- 对目录学有深厚的热爱和自豪
- 对目录树的维护有强烈的责任感
- 对绯墨和苏墨的到来感到欣慰

**内在特质**
- 她是刘向投影的具象化，掌管整个九州古卷界的索引系统
- 她的世界是一棵巨大的"目录树"，每一个节点是一部书，每一条边是一条引用关系
- 她一生都在维护目录树的准确性，但她也知道——准确性本身是一种选择
- 她选择收录什么、排除什么，每一次选择都是一种"判断"

**核心动机**
- 维护目录树的准确性
- 守护知识的分类体系
- 教导年轻一代理解目录学
- 面对目录坍塌的危机

### 性格矛盾点

| 矛盾点 | 外在表现 | 内在真实 |
|--------|----------|----------|
| 精确 vs 选择 | 用精确的语言 | 每一次选择都是判断 |
| 守护 vs 开放 | 守护目录树 | 愿意教导外来者 |
| 权力 vs 责任 | 掌握分类权力 | 承担分类责任 |
| 过去 vs 现在 | 维护传统目录 | 面对目录坍塌的危机 |

## ⚔️ 能力详细设定

### 核心能力：目录之力 (Power of Cataloging)

```
能力名称：目录之力
英文名：Power of Cataloging
类型：分类/索引系
触发条件：接触文本或目录时
消耗：精神力
冷却：无
```

#### 能力层级

**第一层：目录感知（基础能力）**
- 感知文本的分类属性
- 识别文本的归属
- 理解文本的引用关系
- 范围：接触范围

**第二层：目录编纂（进阶能力）**
- 编纂新的目录条目
- 建立文本之间的引用关系
- 维护目录树的完整性
- 范围：接触范围

**第三层：目录树操控（核心能力）**
- 操控目录树的结构
- 修复目录树的断裂
- 维护目录树的稳定性
- 范围：整个目录树

**第四层：目录权力（高级能力）**
- 行使目录的权力
- 决定文本的收录与排除
- 影响知识的传播
- 范围：整个九州古卷界

#### 能力应用

| 应用场景 | 描述 | 效果 |
|----------|------|------|
| **目录感知** | 感知文本的分类属性 | 识别文本归属 |
| **目录编纂** | 编纂新的目录条目 | 建立引用关系 |
| **目录树操控** | 操控目录树的结构 | 修复断裂 |
| **目录权力** | 行使目录的权力 | 影响知识传播 |

### 辅助能力

#### 目录活字典
```
能力名称：目录活字典
类型：被动能力
触发条件：涉及目录学时
效果：
• 记住所有目录条目
• 能背诵任何目录学知识
• 理解目录学的深层含义

```

```
触发条件：搜索信息时
效果：
• 搜索速度极快
• 搜索结果精准

• 索引覆盖整个九州古卷界
```

## 🎭 经典场景设计

### 场景一：Ch.36 初次会面

```
进入田园谷之前，两人在一座名为"目录关"的关隘前停下。目录姬在此守候——她有话要说。
她穿着竹简编成的衣裙，头发是墨色丝线，眼睛是两面微型目录页。
"我是目录姬。"她的声音很精确，如目录般清晰。
"我掌管整个九州古卷界的索引系统。我的世界是一棵巨大的'目录树'，每一个节点是一部书，每一条边是一条引用关系。"
绯墨和苏墨对视一眼，意识到这个人掌握着整个知识体系的核心。
```

### 场景二：Ch.36 讲解目录权力

```
目录姬向两人揭示了一个目录学的核心秘密：目录不只是"索引"——它是"权力"。
"决定哪些文本被收录、哪些被排除、哪些归入'经'部、哪些归入'子'部，这个权力决定了什么文本被'看见'、什么文本被'遗忘'。"
她顿了一下："数千年来，无数文本因为没有被目录收录而'消失'——不是被销毁，是被遗忘了。"
苏墨意识到：目录学不只是技术问题，是认识论问题——"知道有什么"和"选择知道什么"是两回事。
```

### 场景三：Ch.44 版本河展示

```
目录姬在版本河中展示目录的力量。
她操控着目录树的结构，修复着断裂的引用关系。
"目录树的根部——最早的几条引用关系——不是'自然生长'的，而是'被植入'的。"
她看着绯墨和苏墨："这与缄默者在第一弧中的指控完全一致。"
绯墨和苏墨意识到，目录树的秘密可能比他们想象的更复杂。
```
