## 🎨 外貌详细设定

### 发型与发色
- **颜色**：深墨色中夹杂灰白（岁月的痕迹）
- **质感**：干枯、随意
- **长度**：及肩
- **造型**：随意披散，不修边幅

### 面部特征
- **眼睛**：深邃墨色，如竹林深处的幽暗
- **眼神**：沉默、坚定、带着隐忍的智慧
- **特殊**：脸上有很深的皱纹，是岁月和思考的痕迹

### 体型
- **身材**：瘦高
- **体态**：挺拔但微微佝偻，是长期隐居的结果
- **特点**：手指修长，是弹琴和翻阅古籍的痕迹

### 服饰
- **款式**：洗得发白的旧袍
- **颜色**：原本是深灰色，现已褪成灰白
- **配饰**：无
- **鞋子**：磨损的布鞋

### 气味特征
- **主调**：竹子的清新气息
- **中调**：菊花的淡雅香气
- **尾调**：墨水的苦涩味道
- **整体**：让人联想到竹林深处的隐居生活

## 💭 性格详细设定

### 核心性格：沉默的智者

**表面表现**
- 说话极简——与辞章的铺排形成鲜明对比
- 慷慨但隐忍
- 看到绯墨时，琴声停了："你长得像他。"——说的是墨渊
- 对绯墨有特殊的情感（因她是墨渊的养女）

**内在特质**
- 知道墨渊的秘密
- 墨渊选择了"维护"而非"揭露"——竹林客理解这个选择
- 隐居在田园谷，守护着魏晋文学的传承
- 对绯墨和苏墨的到来感到欣慰

**核心动机**
- 守护田园谷的传承
- 保护绯墨（墨渊的养女）
- 传递墨渊的秘密
- 引导绯墨和苏墨理解魏晋文学

### 性格矛盾点

| 矛盾点 | 外在表现 | 内在真实 |
|--------|----------|----------|
| 沉默 vs 慷慨 | 说话极简 | 内心慷慨、愿意分享 |
| 隐居 vs 守护 | 隐居在竹林深处 | 守护着田园谷的传承 |
| 隐忍 vs 坚定 | 看似隐忍 | 内心坚定、不可动摇 |
| 过去 vs 现在 | 回忆墨渊的过去 | 面对现在的责任 |

## 📚 知识承载

### 主要知识领域：建安文学、魏晋风骨、文学自觉

**建安文学知识**
- 建安文学概论：建安风骨——慷慨悲凉、刚健有力
- 建安七子与三曹——文学集团的形成
- 曹操诗歌——"对酒当歌，人生几何"的慷慨之气
- 曹植的文学成就——"骨气奇高，词采华茂"（钟嵘评）

**魏晋风骨知识**
- 魏晋风骨的精神内涵
- 魏晋文学的自觉
- 魏晋文人的生存状态
- 魏晋文学的时代背景

**文学自觉知识**
- 曹丕《典论·论文》——"文以气为主"、文学独立的宣言
- 陆机《文赋》——创作论的先驱
- 刘勰《文心雕龙》——中国文学理论体系的巅峰
- 文学自觉的标志——文学脱离经学附庸地位

## 🎭 经典场景设计

### 场景一：Ch.37 初次会面

```
田园谷是"谷地清幽、涧水曲折"的世外之地。空气中有菊花的气味和淡淡的墨香。
绯墨在这里找到了竹林客——一个隐居在竹林深处的中年文人，穿着洗得发白的旧袍，正在竹林中弹琴。
他看到绯墨时，琴声停了。
"你长得像他。"——他说的是墨渊。
绯墨的眼睛湿润了。
```

### 场景二：Ch.38 讲述建安文学

```
竹林客开始为他们讲述田园谷的历史——也是魏晋南北朝文学的历史。
他用"慷慨悲凉"四个字概括了建安文学的气质。
"曹操的'对酒当歌，人生几何'——那是对生命短暂的感慨。
曹植的'骨气奇高，词采华茂'——那是对文学的追求。
建安七子——那是文学集团的形成。"
他停顿了一下："建安文学，是幸存者的证词。"
```

### 场景三：Ch.39 引导文学自觉

```
竹林客带他们到田园谷的核心——一片被称为"自觉之池"的水面。
池水清澈见底，但水面下有无数文字在流动。
"魏晋时期，中国文学发生了'自觉'——文学从'经学的附庸'变成了'独立的艺术'。"
他指着池水："你看这些文字——它们知道自己是文学。"
绯墨看着池水，感受到了文字的自我意识。
```
