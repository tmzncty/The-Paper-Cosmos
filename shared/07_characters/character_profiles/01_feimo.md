## 🎨 外貌详细设定

### 发型与发色
- **颜色**：绯红色（深桃红/绯红，如陈年朱砂）
- **质感**：微卷，阳光下呈现温暖的琥珀光泽
- **长度**：及腰长发
- **造型**：用墨色发带束起，留出两缕鬓发
- **特殊效果**：情绪激动时发梢会微微发光

### 猫耳特征
- **颜色**：绯红色（内侧粉白色）
- **大小**：中等，与头部比例协调
- **动态**：
  - 专注时微微向前倾
  - 警惕时竖起
  - 开心时轻轻抖动
  - 害怕时向后压平
- **敏感度**：对声音极其敏感，能听到细微的声响

### 猫尾特征
- **颜色**：绯红色（末端白色）
- **长度**：及膝
- **动态**：
  - 专注时轻摇
  - 紧张时缠绕腿边
  - 开心时高高翘起
  - 害怕时夹紧
  - 思考时画圈
- **触感**：柔软蓬松

### 眼睛
- **颜色**：深邃墨色（如纯净墨锭）
- **形状**：猫眼形状，瞳孔可放大缩小
- **特殊效果**：
  - 好奇时映出光芒
  - 专注时深不见底
  - 使用能力时浮现墨色纹路
  - 暗处会发光（夜视能力）

### 体型
- **身材**：娇小纤细
- **体态**：轻盈优雅，动作如猫
- **特点**：
  - 手脚小巧灵活
  - 指尖常年沾染淡淡墨痕
  - 皮肤白皙，透出淡淡绯红

### 服饰

#### 日常司书服
- **款式**：改良式司书服（深色长袍/连衣裙+绯色滚边）
- **颜色**：深靛蓝为主色，绯红色滚边
- **材质**：轻盈透气的布料
- **配饰**：
  - 多口袋围裙（装满书签、小工具）
  - 墨色发带
  - 简约的银色书签项链
- **鞋子**：软底布鞋，方便攀爬书架

#### 工作服
- **款式**：轻便的短上衣+长裤
- **颜色**：灰色+绯色点缀
- **特点**：
  - 袖口可收紧，方便工作
  - 多口袋设计
  - 耐磨材质

#### 特殊场合
- **款式**：正式和服风格的长裙
- **颜色**：绯红渐变到墨色
- **特点**：
  - 有精美的文字刺绣
  - 配有披帛
  - 适合重要仪式

### 气味特征
- **主调**：旧书页的淡淡香气
- **中调**：墨水的清新气息
- **尾调**：阳光的温暖味道
- **整体**：让人联想到安静的图书馆午后

## 💭 性格详细设定

### 双面性格结构

#### 【绯】外在表现层

**社交表现**
- 内向羞涩，认生
- 被注视时脸颊绯红
- 说话轻柔，声音像小猫一样
- 容易害羞，被夸奖时会捂住脸
- 社交恐惧，不擅长与陌生人交流

**日常行为**
- 喜欢躲在书架后面观察别人
- 与人对话时会不自觉地玩弄发梢
- 紧张时会结巴
- 害怕时会"喵呜"一声

#### 【墨】内在核心层

**工作态度**
- 专注沉静，工作时异常专注，雷打不动
- 对书籍有近乎神圣的敬畏
- 对损坏书籍的行为零容忍
- 工作时会展现出完全不同的一面——自信、坚定

**知识追求**
- 对未知充满好奇
- 渴望学习新知识
- 对古籍有深厚的感情
- 愿意为保护知识付出一切

#### 【猫】天性特征

**感官特点**
- 好奇心强，对新事物充满探索欲
- 喜欢温暖，总能找到最舒服的角落
- 感官敏锐，听觉嗅觉超越常人
- 夜视能力强

**行为习惯**
- 受惊时会"喵呜"
- 高兴时会发出"呼噜"声
- 喜欢蜷缩在温暖的地方看书
- 会追逐移动的小物体
- 对毛线球有特殊喜好

#### 【少女】可爱面

**兴趣爱好**
- 偷偷喜欢精美插画故事书
- 收集各种可爱书签
- 会和书本说话，把它们当朋友
- 喜欢在书页间夹花瓣

**情感表达**
- 偶尔会露出孩子气的一面
- 对可爱的东西毫无抵抗力
- 会偷偷画小插画
- 喜欢收集各种小物件

### 性格矛盾点

| 矛盾点 | 外在表现 | 内在真实 |
|--------|----------|----------|
| 羞涩 vs 勇敢 | 平时害羞胆小 | 为保护书籍会鼓起勇气 |
| 内向 vs 热情 | 不善与人交流 | 对知识充满热情 |
| 柔弱 vs 坚强 | 看起来柔弱 | 内心非常坚强 |
| 依赖 vs 独立 | 依赖熟悉的环境 | 能独立面对困难 |

## ⚔️ 能力详细设定

### 核心能力：墨迹通晓 (Ink Resonance)

```
能力名称：墨迹通晓
英文名：Ink Resonance
类型：感知/共鸣系
触发条件：触摸书籍或文字载体
消耗：精神力
冷却：无（但过度使用会导致精神疲劳）
```

#### 能力层级

**第一层：物理感知（初始能力）**
- 感知书籍的材质、年代、保存状态
- 识别墨水成分、书写工具
- 检测书籍的损伤程度
- 范围：触摸范围

**第二层：内容感知（第一卷觉醒）**
- 快速理解书籍内容主旨
- 感知文字背后的情感色彩
- 识别作者的写作意图
- 范围：触摸范围，但理解深度增加

**第三层：历史感知（第三卷觉醒）**
- 感知前任读者的情绪残留
- 窥见书籍经历的重要事件
- 理解书籍在历史中的流转
- 范围：触摸范围，但可感知时间维度

**第四层：深层共鸣（第五卷觉醒）**
- 与书籍建立灵魂连接
- 进入书籍构建的虚拟空间
- 与书中角色对话
- 范围：可远程连接（需曾触摸过）

#### 能力应用

| 应用场景 | 描述 | 效果 |
|----------|------|------|
| **书籍鉴定** | 触摸书籍判断真伪、年代 | 100%准确率 |
| **情感感知** | 感知书中蕴含的情感 | 理解作者/读者心境 |
| **损伤检测** | 发现书籍的隐性损伤 | 提前预防损坏 |
| **历史回溯** | 窥见书籍的历史 | 获取重要信息 |
| **共鸣连接** | 与书籍建立深层连接 | 进入虚拟空间 |

## 💕 重要关系

### 绯墨 ↔ 校书郎

**关系类型**：双主角搭档

**发展阶段**：
1. **陌生期**：绯墨害羞，校书郎迷茫，互相不理解
2. **试探期**：开始交流，发现彼此的不同
3. **信任期**：建立信任，开始合作
4. **默契期**：配合默契，互相依赖
5. **超越期**：超越搭档关系，成为命运共同体

**互动特点**：
- 绯墨教校书郎文渊的知识
- 校书郎帮绯墨走出舒适区
- 两人互补，缺一不可
- 情感从友情逐渐深化

**关系类型**：青梅竹马 + 守护者

**特点**：

**发展**：

## 📚 知识承载

### 主要知识领域：古代文学（文献保护）

绯墨作为司书，承载着大量关于古代文献保护的知识：

**文献载体知识**
- 竹简、木牍的材质与制作
- 帛书的织造与书写
- 纸张的发明与演变
- 各种书写工具的特点

**文献保护知识**
- 书籍的保存环境
- 虫蛀、霉变的防治
- 古籍修复技术
- 文献复制方法

**版本学知识**
- 善本的判定标准
- 版本的流传系统
- 校勘的基本方法
- 辨伪的基本技巧

### 知识传递方式

1. **日常对话**：自然地谈论书籍相关知识
2. **工作展示**：在修复/整理书籍时展示专业知识
3. **能力使用**：使用墨迹通晓时解释相关知识
4. **问题解答**：回答校书郎的疑问
