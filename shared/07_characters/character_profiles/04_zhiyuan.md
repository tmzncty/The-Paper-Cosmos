# 纸鸢 (Zhǐ Yuān) 详细档案

## 📋 基础档案

| 属性 | 描述 |
|------|------|
| **全名** | 纸鸢 |
| **种族** | 文渊原生种族（人类形态） |

## 🎨 外貌详细设定

### 发型与发色
- **颜色**：深褐色（如陈年竹简的颜色）
- **质感**：柔顺、有光泽
- **长度**：及肩
- **造型**：用纸白色发带束起，留出几缕鬓发

### 面部特征
- **眼睛**：琥珀色（如阳光下的旧纸张）
- **眼神**：温暖、笃定、洞察力强
- **特殊**：眼睛会随着情绪变化而变化色调

### 体型
- **身材**：中等偏瘦
- **体态**：轻盈优雅，动作如纸鸢般灵动
- **特点**：手指灵巧，折纸速度极快

### 服饰
- **款式**：纸白色长衫
- **颜色**：纸白色为主，点缀深褐色
- **配饰**：多口袋围裙（装满纸鸢信使和情报工具）
- **鞋子**：轻便的布鞋，方便快速移动

### 气味特征
- **主调**：新纸张的清新气息
- **中调**：墨水的淡雅香气
- **尾调**：阳光的温暖味道
- **整体**：让人联想到晴朗天空下的纸鸢

## 💭 性格详细设定

### 核心性格：温暖的洞察者

**表面表现**
- 温暖、笃定、洞察力强
- 用纸制品存储和传递信息
- 对绯墨有特殊好感（"第一个让我觉得'也许这次是真的'的人"）
- 精通理论但不炫耀，只会"用"

**内在特质**
- 情报网络覆盖整个文渊
- 折纸速度极快，手指灵巧
- 对绯墨有姐姐般的关爱
- 在绯墨面前会放下"工作面具"

**核心动机**
- 维护情报网络的运转
- 保护绯墨和苏墨
- 传递关键信息
- 成为可靠的"姐姐"

### 性格矛盾点

| 矛盾点 | 外在表现 | 内在真实 |
|--------|----------|----------|
| 轻盈 vs 沉重 | 看似轻松自在 | 承载着沉重的情报责任 |
| 温暖 vs 疏离 | 对人温暖友好 | 保持情报商的专业距离 |
| 洞察 vs 沉默 | 看透一切但不说破 | 用行动而非言语表达关心 |
| 自由 vs 束缚 | 如纸鸢般自由 | 被情报网络束缚 |

## ⚔️ 能力详细设定

### 核心能力：纸鸢网络 (Paper Kite Network)

```
能力名称：纸鸢网络
英文名：Paper Kite Network
类型：感知/传递系
触发条件：使用纸鸢信使时
消耗：精神力
冷却：无（但过度使用会导致精神疲劳）
```

#### 能力层级

**第一层：纸鸢信使（基础能力）**
- 操控纸鸢信使传递信息
- 纸鸢信使能飞越文渊的各个区域
- 信息传递速度极快
- 范围：整个文渊

**第二层：情报感知（进阶能力）**
- 通过纸鸢网络感知情报
- 接收来自各个区域的信息
- 分析情报的真实性和重要性
- 范围：整个文渊

**第三层：网络编织（核心能力）**
- 编织覆盖整个文渊的情报网络
- 协调多个情报源
- 整合分散的信息
- 范围：整个文渊

**第四层：预警系统（高级能力）**
- 建立预警系统
- 提前感知危机
- 提供预警信息
- 范围：整个文渊

#### 能力应用

| 应用场景 | 描述 | 效果 |
|----------|------|------|
| **信息传递** | 传递紧急信息 | 速度快、覆盖广 |
| **情报收集** | 收集各个区域的情报 | 全面、及时 |
| **情报分析** | 分析情报的真实性和重要性 | 准确、深入 |
| **预警系统** | 提前感知危机 | 及时、可靠 |

### 辅助能力

#### 折纸大师
```
能力名称：折纸大师
类型：主动能力
触发条件：使用纸张时
消耗：少量精神力
冷却：无
效果：
• 折纸速度极快
• 能折出各种形态的纸鸢
• 纸鸢具有特殊功能
• 纸张在她手中如活物般灵动
```

#### 洞察之眼
```
能力名称：洞察之眼
类型：被动能力
触发条件：与人交流时
消耗：无
效果：
• 能看透他人的真实意图
• 识别谎言和伪装
• 理解他人的内心需求
• 提供精准的建议
```

## 📈 角色弧线

**起始状态**
- 在风雅原与辞赋峰交界地带经营情报网络

- 对绯墨和苏墨的到来感到好奇

**面临挑战**

- 如何保护绯墨和苏墨

**成长目标**
- 建立与绯墨的信任关系
- 提供关键情报支持
- 成为可靠的"姐姐"

**面临挑战**
- 如何处理越来越复杂的危机
- 如何保护绯墨和苏墨
- 如何平衡情报商的身份与朋友的身份

**成长目标**
- 提供关键预警信息
- 保护绯墨和苏墨

**标志性事件**
- Ch.35：发出版本危机的紧急预警

### 第三阶段：守护者（第三弧及以后）

**起始状态**
- 情报网络更加完善
- 与绯墨的关系更加亲密
- 成为绯墨的"姐姐"般的存在

**面临挑战**
- 如何面对更大的危机
- 如何保护绯墨和苏墨

## 💕 重要关系

### 纸鸢 ↔ 绯墨

**关系类型**：姐姐/妹妹

**特点**：
- 纸鸢对绯墨有特殊好感（"第一个让我觉得'也许这次是真的'的人"）
- 绯墨在纸鸢面前会放下"工作面具"
- 纸鸢像姐姐一样关爱绯墨
- 两人之间有深厚的信任

**发展**：
- 从情报商到"姐姐"
- 从信任到依赖
- 最终成为家人

### 纸鸢 ↔ 苏墨

**关系类型**：盟友/朋友

**特点**：
- 纸鸢对苏墨的理论能力感到欣赏
- 苏墨对纸鸢的情报能力感到敬佩
- 两人在情报分析上有共同语言
- 互相尊重、互相支持

**发展**：
- 从盟友到朋友
- 从合作到信任
- 最终成为可靠的伙伴

**关系类型**：情报对象/警惕者

**特点**：

## 🎭 经典场景设计

### 场景一：Ch.14 初次会面

```
纸鸢站在树屋门口，看着绯墨和苏墨走近。
她的眼睛是琥珀色的，温暖而笃定。
"你就是绯墨？"她的声音很轻，像是纸张翻动的声音。
绯墨点点头，有些害羞。
纸鸢笑了："第一个让我觉得'也许这次是真的'的人。"
她从围裙里取出一只纸鸢信使："送给你。以后有事，就用它联系我。"
```

### 场景二：Ch.35 紧急预警

```
纸鸢的纸鸢信使飞进辞赋峰，带来紧急情报。
"版本危机不只在辞赋峰——田园谷和诗唐原野也出现了类似的现象。"
纸鸢的声音很急，但依然保持冷静。
"更令我不安的是，所有出现版本危机的区域，都是'目录树上引用关系最密集'的节点。"
绯墨和苏墨对视一眼，意识到问题的严重性。
```

### 场景三：日常互动

```
纸鸢坐在树屋里，折着纸鸢信使。
绯墨坐在她身边，看着她灵巧的手指。
"纸鸢姐姐，你为什么对我这么好？"
纸鸢停下手中的动作，看着绯墨："因为你让我觉得，也许这次是真的。"
她继续折纸："在这个世界里，很少有人能让我有这样的感觉。"
```
