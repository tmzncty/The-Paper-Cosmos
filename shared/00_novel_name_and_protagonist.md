# 《纸上宇宙》命名与主角设定

## 小说名字
- **中文**：纸上宇宙
- **英文**：Paper Universe
- **含义**：文学即宇宙，纸上世界比现实更广阔。简洁，有现代感，不容易重名。

---

## 主角一：绯墨（Fēi Mò）

### 基本信息
| 项目 | 内容 |
|------|------|
| 种族 | 猫娘 |
| 职业 | 古籍图书馆司书 / 知识守护者 |
| 外表年龄 | 15-17岁（少女阶段） |
| 名字含义 | 绯=外在羞涩温暖，墨=内在对知识的深沉热爱 |

### 外貌特征

**头发**：
- 绯红色（深桃红/绯红，如陈年朱砂），不是艳俗的粉
- 柔软蓬松，自然微卷
- 用简单墨色发带束起

**猫耳与尾巴**：
- 与发色一致的绯红色
- 耳朵内侧粉白色，对声音敏感，情绪波动时微微抖动
- 尾巴长而蓬松，末端一小撮白色
- 专注时无意识轻摇，紧张时紧紧缠绕腿边

**眼睛**：
- 深邃墨色眼瞳，如纯净墨锭
- 好奇/兴奋时眼中映出光芒（墨池中投入星子）
- 长睫毛微微下垂，增添羞怯感

**身材与着装**：
- 娇小纤细
- 改良式司书服：深色（墨色/靛蓝）长袍或连衣裙，绯色滚边/纽扣
- 多口袋围裙（借书卡、羽毛笔、放大镜、备用墨锭）
- 偶尔戴无度数圆形复古眼镜

**其他细节**：
- 指尖常年沾染淡淡墨痕
- 身上有旧书页、墨水和阳光晒过的味道

### 性格特点

| 维度 | 表现 |
|------|------|
| **内向与羞涩（绯）** | 认生害羞，不擅长与陌生人交流。被注视/称赞时脸颊绯红，猫耳微微发烫。说话轻柔怯生生 |
| **专注与沉静（墨）** | 工作/阅读时异常专注宁静。对书籍和知识有近乎虔诚的热爱和尊重。在熟悉环境中可靠沉稳 |
| **责任感强** | 对司书职责非常认真。为保护书籍会鼓起勇气，哪怕面对害怕的事物 |
| **猫的天性** | 好奇心强（尤其奇怪书籍/传说）。喜欢温暖安静角落（阳光窗台/壁炉边）。听觉嗅觉敏锐。受惊时弓身、尾巴毛炸开、发出很小"喵呜" |
| **少女心** | 偷偷喜欢精美插画故事书，收集可爱书签。对浪漫故事好奇但自己无经验 |

### 能力与技能

| 能力 | 描述 |
|------|------|
| **墨迹通晓** | 对墨水和文字有特殊感应。触摸感知书籍内容、年代、前任读者情绪残留。修复古籍、解读褪色/加密文字有独特天赋 |
| **图书馆活地图** | 对图书馆了如指掌，最偏僻角落和最冷门分类都能准确找到 |
| **超凡平衡感与灵活性** | 猫娘天赋，轻松攀爬高高书架，穿梭狭窄书堆，动作轻盈无声 |
| **微弱"言灵"亲和** | 小声阅读特定书籍段落时可产生微弱现实影响（让枯萎植物恢复生机、暂时安抚烦躁人心），自己未完全意识到或无法自由控制 |

## 主角二：校书郎

**基础设定**（符玄世界观框架）：
- 来源：现代文学研究生/考研生，因意外进入文渊
- 身份：校书峰见习校书郎
- 特殊能力：跨域共鸣——天生能感知不同文脉之间的互文关系
- 成长线：录文生→注疏士→校勘师→阐发者→立论家

**与绯墨的关系**：
- 双主角，互补型搭档
- 绯墨：文渊原住民，守护知识
- 校书郎：外来者，带来跨域视角
- 关系发展：从陌生到信任（非恋爱套路）

---

## 设计原则
1. 绯墨是读者的"代入视角"——通过她的眼睛了解文渊
2. 校书郎是"推动力"——他的跨域能力推动剧情前进
3. 两人的互动是知识与情感的交织
4. 绯墨的成长线：从躲在图书馆到勇敢面对世界
