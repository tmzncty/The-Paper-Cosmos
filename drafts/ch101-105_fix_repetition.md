## ch102 特别说明（最严重章节）

ch102 "是最" 原始密度 10.63‰ 为本批最高（113次/10631字）。核心问题是全文大量出现 "X是最好的。" 级联模板——同一短语 "统一是最好的" 在文中出现约10次，"放弃自己是最可悲的" 出现5次，"第一步是最重要的" 出现4次。

**修复策略**：
1. 保留各段落首次出现的结构性"是最"（如"为什么人的问题是最根本的问题"、知识清单等）
2. 对重复短语轮换替代方案：`统一才完整` / `统一才是正道` / `统一才有力量` / `统一是关键`
3. "放弃自己是最可悲的" → `何其可悲` / `是可悲的` / `最为可惜` / `是真正的遗憾`
4. "行动是最终的" → `是归宿` / `才是目的` / `是落脚点`
5. 长链式级联（如 L205 新秧歌特点段）整体重写短语

**修复后仍保留的10个"是最"**：全部为结构性关键位置（如"交响乐是最丰富的"、"为什么人的问题是最根本的问题"、知识清单中的"是最根本的问题"等），不再构成重复。

## ch105 特别说明

ch105 "是最好的" 原始出现38次，为五章之最。核心结构是胡同三段式：
> "X是最好的。最好的Y是Z。"

修复为多样化替代：`是极好的` / `再好不过` / `恰到好处` / `才最可贵` / `是好的` / `无可替代` 等20种轮换，避免形成新的重复模式。
