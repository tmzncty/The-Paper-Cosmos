### 5.2 具体修复方案

#### P0：「是最」—— 系统性重构（755处）

**目标**：每章降至 ≤ 20次（≤ 3次/千字）
**削减幅度**：约 73%

**替换策略**（按语境选择）：

| 原句式 | 替换方式 | 示例 |
|--------|----------|------|
| "X是最好的" | 删除判断，保留描写 | "谦虚是最好的入门姿态" → "谦虚地走了进去" |
| "X是最Y的" | 降级为"X很Y"或用具体描写替代 | "命是最沉重的" → "命落在纸上，比石头更沉" |
| "X是最Y的。Y够了。" | 删除整组，用行为推进叙事 | 全删，接下一句行动 |
| 连续多个"是最" | 保留1个，其余改为动词/比喻 | 一段3个"是最"→留最有力的1个 |

**处理顺序**：先修 ch100（13.28）、ch102（12.60）、ch105（11.86），这三个章节是密度峰值。

#### P1：「纯够了」—— 段尾清扫（140处）

**目标**：ch99 降至 ≤ 8次，ch100 降至 ≤ 8次
**削减幅度**：约 89%

**替换策略**：
- "XX够了。"→ 直接删除，段落自然结束
- 如果"够了"承担叙事推进功能 → 替换为具体动作："够了。他站起来。" → "他站了起来。"
- 如果"够了"是角色台词的语气词 → 保留（但全章不应超过 5 次）

#### P2：「就够了。」—— 收束句改写（156处）

**目标**：ch96 降至 ≤ 15次，ch97 降至 ≤ 10次，ch98 降至 ≤ 8次
**削减幅度**：约 79%

**替换策略**：
- "XX就够了" → 删除，直接用下一段开头
- "前进就够了" → "继续走。"
- "保存就够了" → "留了下来。"
- 保留少量（≤5次/章）作为文体特色，但不能每段都用

#### P3：「不需要」—— 局部调整（38处，仅 ch96）

**目标**：ch96 降至 ≤ 15次
**替换策略**：
- "不需要A。只需要B。" → 合并为"只需B"
- "权利不需要争取。权利是天生的。" → "权利是天生的。"
