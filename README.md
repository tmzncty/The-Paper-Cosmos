## 核心要求

1. **全覆盖**：每个知识点都要涉及到，不能遗漏
2. **可读性**：首先要是一部好看的轻小说
3. **准确性**：知识点不能出错，要能用于考研复习
4. **双语版**：最终要有中文版和英文版
5. **反套路**：不能让人看烦，要出人意料的剧情
6. **思想深度**：不是简单的爽文

- 种族：猫娘
- 职业：古籍图书馆司书 / 知识守护者
- 特征：绯红发、墨色瞳、内向羞涩但为保护知识会鼓起勇气
- 能力：墨迹通晓、图书馆活地图、微弱言灵

### 陆辞渊（Lu Ciyuan）
- 来源：现代考研生穿越
- 身份：校书峰见习校书郎
- 名字含义：辞=文学，渊=文渊——"以辞入渊"
- 能力：跨域共鸣
- 成长线：录文生→注疏士→校勘师→阐发者→立论家

## 世界观核心

- **世界名**：文渊（Architext Realm）
- **核心规则**：文本即现实，阐释即能源
- **五大文力轴**：词章、叙事、理论、考据、跨域共振
- **空间**：九州古卷界 + 百年新声界 + 欧陆剧场界 + 学理峰 + 天问之渊
