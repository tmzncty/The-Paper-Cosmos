# 📖 纸上宇宙 · The Paper Cosmos

> *"万物皆文本，现实由阐释维持。"*

**《纸上宇宙》**不是一部普通的小说——它是一座伪装成轻小说的文学考研知识库。

120 章主线 + 12 部番外 + 106 张冲刺卡片，将 **4526 条考研知识点**（中国古代文学史、现代文学三十年、当代文学史、文学理论教程、文献学讲义、欧洲文学史、CCTalk 课程笔记）编织进一个「文本即现实」的异世界冒险中。

读完它，你不仅读完了一部完整的长篇轻小说，也复习完了文学考研的全部核心考点。

---

## 🌌 世界观

**文渊（Architext Realm）**——一个由文本、阐释和互文关系构成的世界。这里的山川不是岩石，而是被千年阅读压缩的纸浆；这里的城邦不是砖石，而是历代学者对经典的层层注疏；这里的灾厄不是地震洪水，而是误读与断章取义所化的「误兽」。

五位一体的文力体系——**词章、叙事、理论、考据、跨域共振**——不只是魔法，更是文学研究能力的映射。主角从「录文生」到「立论家」的成长，就是考研备考的完整进路。

---

## 🎭 主角

### 绯墨
文渊原住民，古籍图书馆的猫娘司书。绯红发、墨色瞳，内向羞涩，却在守护知识的时刻爆发出惊人的勇气。她是这个世界最温柔的锚——所有文本的守护者，也是穿越者的引路人。

### 苏墨
现代考研生，穿越到文渊成为一名见习校书郎。"墨"字与绯墨呼应，名字本身就是"书写/知识"的隐喻。他从零开始学习文渊的一切——这不只是在学魔法，而是在学文学考研的全部内容。

## 🎯 设计原则

1. **全覆盖**——每一条知识点都在故事中有对应落点
2. **先是一部好小说**——可读性是第一位的，知识点是自然融入而非生硬灌输
3. **准确性**——所有文学史实、理论概念均经过多轮审校
4. **双语版**——最终产出中文版 + 英文版
5. **反套路**——拒绝爽文模板，追求叙事惊喜与思想深度
