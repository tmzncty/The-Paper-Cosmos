## 三、废话/重复

### 3.1 苏墨考点版与绯墨深度版大面积重叠

**性质**：这是番外12的结构设计——每章绯墨在山腰做深度解读，苏墨在山脚做考点整理。重叠是故意的。

**但问题在于重叠程度**：

| 章节 | 重叠内容 | 严重度 |
|------|---------|--------|
| Ch.1 | 绯墨得出"一体四面，四者互相制约"→苏墨考点版第3条逐字复述 | ⚠️中 |
| Ch.2 | 绯墨"物→情→志→诗"链条→苏墨时间线五点的内容几乎逐条对应 | ⚠️中 |
| Ch.3 | 绯墨三组分类→苏墨未直接复述，而是用"追问之路"收束 | ✅可 |
| Ch.4 | 绯墨"汪琬的量尺逻辑"→苏墨"我的评价"段落与绯墨高度相似（苏墨自己承认"你把我的笔记抄下来了"） | ⚠️中 |
| Ch.5 | 绯墨"套板/转化"→苏墨考点版第3-4条复述 | ⚠️中 |
| Ch.6 | 绯墨"四步辩护+局限"→苏墨刻在石阶上的条目几乎一模一样 | ❌重 |

**判定**：Ch.6重叠最严重——苏墨在L261-263几乎逐字复述了绯墨的局限分析。但Ch.6通过"你在上面理解王逸。我在下面理解你。一样的动作。以意逆志。"为这种重叠赋予了叙事意义。**建议**：不需大改，但应确保苏墨版每章至少有一个独立观察（不是绯墨内容的简写版），目前Ch.3较好（苏墨有自己的"追问之路"表述），Ch.1/2/4/5/6偏弱。

### 3.2 墙面反馈模板化

六章墙面反馈的核心模板高度重复：

> 墙面温度变化（凉→温→回暖）→ 字迹/光带变化（浮现→稳定→收回）→ 墙面状态（平静/安静/宁静/完成）

|---------|------|------|------|------|------|------|
| "墙面变暖了" | ✓ | — | ✓ | — | ✓ | — |
| "温度回来了" | ✓ | ✓ | — | ✓ | — | — |
| "光全部收回/收进" | ✓ | ✓ | ✓ | ✓ | ✓ | — |

虽然六章中只有三章出现"墙面变暖了"（Ch.1/3/5），但墙面反馈的基本叙事弧线六章完全一致。Ch.6稍有变化（改用石柱震动、场景变化替代墙面温度），值得肯定。

**判定**：模板化中等偏重。不影响阅读流畅性，但建议后续番外减少对"温度变化→光芒收回→墙面安静"这个三段式的依赖。**不要求本轮修改**。
