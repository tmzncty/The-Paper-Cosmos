## 五条核心关联链（论述题答题路径）

**路径1：中西文论对照**
A-07（意境）↔ A-08（典型）↔ J-03（中西文论对比）↔ I-10（失语症）

**路径2：文学思潮演变**
古典主义(E-01) → 浪漫主义(F-01) → 现实主义(F-03) → 自然主义(F-09) → 象征主义(F-10) → 现代主义(G-01) → 后现代(G-05)

**路径3：叙事学理论谱系**
俄国形式主义(H-05) → 结构主义(H-06) → 热奈特(H-07) → 巴赫金(H-11) → 后结构(I-01/02)

**路径4：读者/作者/文本的权力转移**
作者中心(A-01表现说) → 作品中心(A-01客观说) → 读者中心(H-01接受美学) → 作者之死(I-04) → 作者功能(I-08)

**路径5：中国现当代文学的"主义"之旅**
写实(C-03文学研究会) → 浪漫(C-03创造社) → 现代(C-06新感觉派) → 解放(C-10伤痕) → 先锋(C-07) → 寻根(C-08)

## 考研出题方向
